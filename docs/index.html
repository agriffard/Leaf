Leaf

Leaf is a clean Blazor wrapper for Leaflet and MapLibre with support for markers, clustering, popups, and draw tools.

Install

dotnet add package Leaf

Basic usage

<LeafMap Engine="LeafMapEngine.Leaflet"
         Latitude="48.8566"
         Longitude="2.3522"
         Zoom="6"
         Markers="markers"
         EnableClustering="true"
         EnableDrawTool="true" />

Sample app

Run dotnet run --project samples/Leaf.SampleApp to validate interactions locally.
